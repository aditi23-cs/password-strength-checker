# Password Strength Checker 🔐

A Python-based tool that evaluates the strength of a password based on length, complexity, and common password detection.

## 📌 About The Project
This tool analyzes passwords and gives instant feedback on how strong or weak they are, along with suggestions to improve them.

## 🎯 Key Features
- Detects common/weak passwords instantly
- Checks for uppercase and lowercase letters
- Checks for numbers and special characters
- Gives a strength score out of 6
- Provides improvement suggestions

## 🛠️ Technologies Used
- Python 3

## ▶️ How To Run
```bash
python password_checker.py
```

## 📊 Strength Levels
| Score | Strength |
|-------|----------|
| 0-2 | 🔴 Weak |
| 3-4 | 🟡 Moderate |
| 5 | 🟢 Strong |
| 6 | 💪 Very Strong |
